# Git Alias

```
git status

a good alias for it is:
git sta

create alias:
git config --global alias.sta "status"
```

```
a shorter version of git status
git status -s

a good alias for it is:
git st

create alias:
git config --global alias.st "status -s"
```

```
git commit

a good alias for it is:
git co

create alias:
git config --global alias.co "commit"
```

```
git switch

a good alias for it is:
git sw

create alias:
git config --global alias.sw "switch"
```

```
git checkout

a good alias for it is:
git ch

create alias:
git config --global alias.ch "checkout"
```

```
removes files from staging area
git reset HEAD

a good alias for it is:
git rehe

create alias:
git config --global alias.rehe "reset HEAD"

note: 
git cechkout . 
will delete files that are not staged

git clean df
will delete untracked files
```

```
git pull --reset

a good alias for it is:
git pure

create alias:
git config --global alias.pure "pull --reset"
```

useful commands
```
will show all branches, also the remote ones
git branch -a
```
