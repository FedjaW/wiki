# LazyVim Shortkeys

Open mason: `<leader> cm`

Hover Documentation: `K bzw. (shift k)`

Search for a string: `<leader> sg`

View Diagnostics: `<leader> xx`

Change colorscheme: `<leader> uC`

Toggle between buffers: `<leader fb>`

Go to next buffer (tab): `[b`

Go to prev buffer (tab): `]b`

Vertical split: `<leader> |`

Horizontal split: `<leader> -`

Help page: `<leader> sh`

Search keymaps: `<leader> sk`

Search and replace (spectre plugin): `<laeder> sr`

Open lazygit: `<leader> g`

Open terminal: `<leader> ft`
